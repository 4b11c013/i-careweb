USER INTERFACE GUIDE

互動介面操作流程

01

準備階段 Preparation

在開始餵食前，系統會引導照護者完成必要的衛生準備工作，確保環境安全。

  • 畫面提示徹底洗手並佩戴手套
  • 確認餵食工具與溫水已備齊
  • 勾選確認清單 (Checklist) 以進入下一步
02

姿勢調整 Positioning

正確的姿勢是預防吸入性肺炎的關鍵，透過圖像化指引，協助照護者調整病患位置。

  • 圖示引導將上半身抬高 30～45°
  • 警示標語：避免平躺狀態下餵食
  • 確認頸部支撐舒適度
03

前置檢查 Checking

系統強制進行安全確認，防止管路滑脫或食物溫度不當造成的風險。

  • 輸入或確認鼻胃管刻度是否移位
  • 提示進行胃反抽檢查 (Gastric Residual)
  • 確認食物溫度（建議滴在手背測試）
04

進行餵食 Feeding

進入主要餵食程序，介面提供計時器與流速建議，確保餵食過程平穩。

  • 顯示建議流速：利用重力慢慢灌入
  • 紅色警示：避免快速推注
  • 緊急暫停按鈕：若病患有嗆咳反應可立即記錄
05

清潔紀錄 Cleaning

餵食結束後的收尾工作，系統會自動記錄本次餵食時間與狀況。

  • 提示用溫水沖洗管路 (Flushing)
  • 清潔推車桌面與灌食器
  • 自動生成照護日誌，可選填病患反應
